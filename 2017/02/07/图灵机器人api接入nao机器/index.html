将图灵机器人api接入nao机器人

发表于 2017-02-07   |   分类于 Python   |  

用nao自带的choregraphe新建项目，结构如下图所示：

简单来说，一共分为4步

  1. 启动nao自带的录音程序，拍打nao头部启动录音，并输出录音文件所在的路径。
  2. 因图灵机器人暂时只支持纯文本输入，需要将录音转换成文字。
  3. 将转换出来的文字调用图灵机器人api，输出回答。
  4. 根据不同回答配合不同肢体动作。

具体类型区别详见官方文档。
